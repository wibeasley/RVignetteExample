Goals of this Pacakge

This vignette is a minimal exmaple of an R package that has a nonstandard vignette. I would like to have a working exmaple of a vignette with these features:

  1. Uses knitr instead of Sweave to build the vignette.
  2. Uses knitr's code externalization feature, where almost all the R code lives in a dedicated *.R file.
  3. Is not rebuilt every time the package is checked. I am pretending this vignette takes a long time to run and compile, and sould be run on CRAN or win-builder computer (or even run on the developer's computer is they don't want to take 5+ minutes)
  4. Saves images to a dedicated directory to make it easier to track changes in GitHub. Presumably this directory is ./vignettes/FigureRmd.

I chose and Rmd file instead of a Rnw, so that it could be run on more development boxes, and increase the chance of getting a more helpful & portable response.
